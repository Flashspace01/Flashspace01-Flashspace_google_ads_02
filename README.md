# FlashSpace Frontend

This folder contains the React + Vite frontend for the FlashSpace landing pages.

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Local development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Environment

Copy `.env.example` to `.env` and configure:

- `VITE_API_BASE_URL`
- `VITE_LEAD_API_KEY` (optional)

## Backend integration

Lead capture API is in `../backend`.
